# Social Media Engagement Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a social media dataset to understand patterns in engagement, sentiment, and user activity.

## Objectives
• Analyze social media engagement metrics  
• Identify states with highest social activity  
• Study relationship between followers and engagement  
• Analyze sentiment distribution  
• Identify viral posts and engagement patterns  

## Tools Used
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Jupyter Notebook  

## Dataset
Social media dataset containing engagement metrics, sentiment features, follower counts, and post information.

## Key Analysis
The project includes:

• Engagement metrics distribution  
• State-wise activity analysis  
• Content type comparison  
• Followers vs Engagement relationship  
• Sentiment analysis  
• Posting activity trends  
• Correlation analysis  
• Viral posts identification  

## Key Insights
• Engagement is highly skewed with few viral posts generating most interactions  
• States like MP and Kerala show higher engagement levels  
• Information-based posts generate slightly higher engagement  
• Accounts with more followers tend to produce higher engagement
